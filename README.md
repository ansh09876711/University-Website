# 🎓 Edustack University Website

An AI-powered university website with chatbot support.

## 🚀 Features
- AI chatbot for student queries
- Course recommendation system
- Placement information
- Admission guidance
- University prospectus AI assistant

## 🛠 Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- Groq AI (Llama 3)
- Three.js animations
